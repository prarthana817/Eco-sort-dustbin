Eco Sort Dustbin

Smart Waste Sorting with Arduino

How it Works

The Eco Sort Dustbin uses an ultrasonic sensor to detect when waste is thrown. The Arduino microcontroller then identifies the type of waste and sorts it automatically using a servo motor. This reduces human effort and promotes efficient waste management.

Features

  • ✅ Automatic waste detection
  • ✅ Arduino-based smart sorting
  • ✅ Eco-friendly and cost-effective
  • ✅ Easy to build and maintain
