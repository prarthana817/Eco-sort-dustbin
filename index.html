Smart Dustbin Using Arduino

Introduction

The Smart Dustbin automatically opens its lid when it detects a person nearby, ensuring hygienic waste disposal using Arduino and sensors.

Components

  • Arduino Uno
  • Ultrasonic Sensor (HC-SR04)
  • Servo Motor
  • Dustbin with Hinged Lid
  • Jumper Wires
  • Battery or USB Power

Working Principle

When the ultrasonic sensor detects an object within 20 cm, the Arduino signals the servo motor to open the lid. After 5 seconds, it closes automatically.

Advantages

  • Touchless waste disposal
  • Improved hygiene
  • Cost-effective automation
  • Can be solar powered

Applications

  • Homes
  • Hospitals
  • Offices
  • Public places
